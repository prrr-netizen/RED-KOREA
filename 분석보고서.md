# 📊 프로젝트 종합 분석 보고서

## 1. 프로젝트 개요

이 프로젝트는 **"Cloud Vend Web" / "HOLYSHARRY.CO"** 브랜드의 **멀티테넌트 웹 쇼핑몰 SaaS 플랫폼**입니다.  
Flask 기반의 Python 백엔드로 구성되어 있으며, 각 가맹점(shop)이 서브도메인(`<shopname>.holysharry.co`) 형태로 독립적인 쇼핑몰을 운영할 수 있게 해주는 시스템입니다.  
Android APK(`자충.apk`)는 실시간 은행 입금 알림을 수신하는 **PushBullet 연동 앱**입니다.

### 📄 push.py (16KB)
**실시간 은행 자동 입금 처리 서버**입니다.

#### 동작 방식
1. 쇼핑몰에서 `/api` 엔드포인트로 POST 요청
2. **PushBullet WebSocket**(`wss://stream.pushbullet.com/websocket/`)에 연결
3. **카카오뱅크** 또는 **토스** 앱 알림을 실시간 수신
4. 입금자명 + 금액이 `bankwait` 테이블과 일치하면 자동 충전 처리
5. Discord Webhook으로 입금 알림 전송
6. push.db의 `getresult` 테이블에 처리 결과 저장 (중복 처리 방지)

#### 지원 은행 앱
- `com.kakaobank.channel` (카카오뱅크)
- `viva.republica.toss` (토스)

```
[Android 앱 (자충.apk)]
        ↓ PushBullet WebSocket

        ↓ Discord Webhook

        ↓ SQLite (각 shop.db, license.db, ban.db)
        ↓ Twilio SMS API
        ↓ Discord Webhook
[Nginx (default 설정)]
        ↓
[holysharry.co / *.holysharry.co]
```
